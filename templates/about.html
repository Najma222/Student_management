About This Project

This course management system was designed to simplify the process of managing academic tasks for both students and instructors. With features like course enrollment, assignment submissions, grading, and user management, it brings together everything needed for an organized learning experience.

Developed using Flask and MySQL, the system aims to provide a clean, intuitive interface along with practical tools to support academic workflows efficiently.
